# Anagram

## Anagram resolver by Dictionary in NodeJS

## Usage

### Console

* npm install
* anagram w "keyword"

>*ex:*
*anagram w roma*

*output =>*

>_Your word is **roma**_


>_{ **result**: ['roma', 'ramo', 'mora', 'maro', 'amor' ], **total**: 5 }_

### Dictionary is updated with each search automatically
